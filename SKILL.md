### API数据结构

从页面 `__NEXT_DATA__` 中提取 `recommendPageData`：

```json
{
  "todayTotal": {
    "buy": 9,
    "sell": 0
  },
  "maxProfit": 0.0432,
  "updateAt": 1773184973222,
  "recommends": [
    {
      "symbol": "DEGO",
      "direction": "buy",
      "maxProfit": 0.0432,
      "updatedAt": "2026-03-11T00:00:10"
    }
  ]
}
```

### 字段说明

| 字段 | 说明 |
|------|------|
| todayTotal.buy | 今日买入信号数量 |
| todayTotal.sell | 今日卖出信号数量 |
| maxProfit | 最大预期盈利（小数形式） |
| updateAt | 更新时间戳（毫秒） |
| recommends | 推荐币种列表 |
| direction | 信号方向 (buy/sell) |
| maxProfit | 预期最大盈利 |
