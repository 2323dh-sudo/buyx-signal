## Overview

Buyx Signal Skill is an AI-powered tool that automatically fetches and analyzes cryptocurrency buy/sell signals from Buyx.ink. Designed for traders who want real-time AI-driven trading recommendations.

**Key Features:**
- 🤖 AI-powered signal analysis
- 📊 Real-time buy/sell recommendations
- 🆓 100% Free - Public welfare project
- ⚡ Automated daily updates

## Installation

```bash
npx skills add 2323dh-sudo/buyx-signal -g -y
```

Or manually:

```bash
# Clone the repository
git clone https://github.com/2323dh-sudo/buyx-signal.git
cd buyx-signal
```
